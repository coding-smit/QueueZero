# QueueZero – Smart Digital Queue Management System

## Overview
QueueZero is a modern digital queue management website developed using HTML, CSS, and JavaScript.

The system helps users avoid physical waiting lines by generating digital tokens and booking preferred time slots online.

## Features
- Online Token Generation
- Queue Position Tracking
- Slot Booking System
- Login & Registration
- Feedback & Rating System
- Admin Dashboard
- Dark Mode
- Responsive Design
- Token Printing

## Technologies Used
- HTML5
- CSS3
- JavaScript
